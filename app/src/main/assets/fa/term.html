1. مقدمه

به برنامه بازیابی رسانه WhatsApp خوش آمدید. با دانلود و استفاده از این برنامه، شما موافقت می‌کنید که با این شرایط استفاده را رعایت کنید. این شرایط استفاده از استفاده شما از برنامه و هرگونه خدمات ارائه شده توسط آن پیروی می‌کند.

2. استفاده از برنامه

شما می‌توانید برنامه بازیابی رسانه WhatsApp را تنها برای بازیابی فایل‌های رسانه‌ای از برنامه WhatsApp خود استفاده کنید. شما نباید از برنامه برای هیچ فعالیت غیرقانونی، غیرمجاز یا غیراخلاقی استفاده کنید.

3. حریم خصوصی

حریم خصوصی شما برای ما مهم است. ما برخی اطلاعات را طبق سیاست حریم خصوصی خود جمع‌آوری و ذخیره می‌کنیم. با استفاده از برنامه، شما به جمع‌آوری و استفاده از اطلاعات شما طبق سیاست حریم خصوصی موافقت می‌کنید.

4. مالکیت فکری

برنامه بازیابی رسانه WhatsApp و تمام محتوای آن، شامل اما محدود به متن، گرافیک، تصاویر، لوگوها و نرم‌افزار، متعلق به توسعه‌دهندگان برنامه است و تحت قوانین مالکیت فکری حفاظت می‌شود. شما نمی‌توانید بدون مجوز صریح کپی کنید، توزیع کنید، تغییر دهید یا کارهای مشتق ایجاد کنید.

5. معافیت از ضمانت

برنامه بازیابی رسانه WhatsApp "به همان شکلی که هست" بدون هرگونه ضمانت یا تعهدی ارائه می‌شود. ما دقت، قابلیت اطمینان یا دسترسی برنامه یا خدمات آن را ضمانت نمی‌کنیم. شما برنامه را به مسئولیت خود استفاده می‌کنید.

6. محدودیت مسئولیت

در هیچ حال، توسعه‌دهندگان برنامه بازیابی رسانه WhatsApp مسئول هیچ گونه خسارت مستقیم، غیرمستقیم، اتفاقی، تبعیضی یا ویژه ای که از استفاده از برنامه ناشی می‌شود، نخواهند بود، حتی اگر از امکان وقوع چنین خسارت‌هایی آگاهی داشته باشند.
